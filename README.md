> File beautifier for Unibeautify

## Installation

```bash
npm install --global @unibeautify/beautifier-file
```

Then change `.unibeautifyrc.yml`:

```yaml
Python:
    beautifiers: ["File"]
```

## Usage

See [`unibeautify-cli`](https://github.com/Unibeautify/unibeautify-cli) for details.
